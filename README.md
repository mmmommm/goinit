# goinit
Generate the initial configuration file for Go.

Here are generated by this cli tool
```
- .github/workflows/ci.yaml
- README.md
- LICENSE
- .gitignore
- .golangci.yml
```

## Usage
`goinit github.com/mmmommm/goinit`

#### Option
`-p ${package_name}`
Run `go mod init ${package_name}`

`-d`
Create dockerfile for Go.
